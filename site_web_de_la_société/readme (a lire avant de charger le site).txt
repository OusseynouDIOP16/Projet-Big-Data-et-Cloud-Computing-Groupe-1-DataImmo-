## Mise en place de l'application (vous devez d'abord installer Xampp)

1- Mettez ce dossier (laplaceImmo) dans le dossier htdocs qui se situe dans le dossier xampp créé lors de l'installation.

2- Lancez le serveur xampp (Apache et MySQL)

3- Tout est ok maintenant, tapez sur votre navigateur : localhost/laplaceImmo/index.html

Merci
